# more-items
A PMDO mod that adds a variety of items into the game!

## Added Items
* Absorb Bulb
* Air Balloon
* Blue Portal Orb
* Cell Battery
* Eviolite
* Fickle Specs
* Lucky Egg **(\*\*)**
* Orange Portal Orb
* Monster Orb
* Protective Pads **(\*)**
* Shiny Charm **(\*\*)**
* Snowball
* Weakness Policy **(\*)**
* All Gem Types
